Виды синтаксиса
    Стандартов синтаксиса, как и словарей, несколько. Разберем наиболее распространенные и не устаревшие:
  • Microdata — Микроданные (словарь Schema.org чаще всего встречается именно в этом синтаксисе);
  • JSON-LD — расширение JSON.
Микроданные
Микроданные (Microdata) — это один из способов помечать данные на веб-страницах, чтобы облегчить их понимание компьютерами, в том числе поисковыми системами.
Ключевые элементы микроданных — это атрибуты itemscope, itemtype и itemprop, указывающие на сущности и их свойства.

— itemscope указывает парсеру на то, что в определенном HTML-блоке содержатся данные об объекте. Часто уточняют, нужно ли указывать значение артибута itemscope — в соответствии со спецификацией HTML5 значение для логических атрибутов задавать не обязательно, достаточно просто указать название атрибута.
— Атрибут itemtype в HTML указывает адрес словаря, который будет применяться для определения свойств элемента в структуре данных. Указывается конкретный тип объекта будь то Person, Organization или Product.
— itemprop указывает на свойства этого объекта. Свойство определяется словарём данных. К примеру, на Schema.org, которую поддерживает Яндекс и Google, перечислены словари с их возможными свойствами.
JSON-LD
JSON-LD - передачи информации, реализованный с использованием текстового формата JSON в JavaScript. JSON-LD используется, когда необходимо указать объекты и подробно раскрыть их свойства. Обычно синтаксис применяют для разметки в рамках словаря Schema.org, а за счет JavaScript эта информация становится читаемой для компьютера. Появление этого метода было настоящим прорывом: у разработчиков появился синтаксис, который, в отличие от других, позволил помещать элементы не в код HTML-страницы, а сразу в скриптовый тег — script. Он быстро стал одним из самых распространенных форматов синтаксиса для разметки контента.
    Обладает рядом преимуществ, например:
  • содержит меньше кода, чем microdata или RDFa, проще воспринимается;
  • рекомендован Google в качестве основного способа микроразметки;
  • для популярных CMS есть плагины, которые позволяют быстро внедрить микроразметку;
  • разметка внедряется посредством JavaScript и видна только поисковым роботам;
  • возможность использовать различные генераторы коды, такие как technicalseo.com или schemaapp.com, что значительно упростит процесс внедрения микроразметки на сайт.
